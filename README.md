# またごち ![Tier](https://img.shields.io/badge/tier-hobby-blue)

> 家族の「また食べたい」を残す、ごはんメモ

## これは何？

またごちは、SNSで見つけたレシピ動画を保存して、作ったら家族の星評価を記録する静的MVPです。
「また作る？」がひと目でわかる、忙しい家庭向けの軽いごはんメモです。

## こんな時に使えます

- SNSで見つけたレシピ動画をあとから探し直したいとき
- 家族ごとの「また食べたい」度を残したいとき
- レシピアプリほど重くない、家庭用のごはんメモを試したいとき

## 使い方

- デモ: https://165cm.github.io/matagochi/
- レシピ名、ショート動画リンク、キャプション、自分用メモを入力して保存します。
- 作ったレシピは「評価」画面で家族ごとの星評価と次回希望を記録できます。

## GitHub Pagesでできること

- Instagram / Facebook / TikTok / YouTube Shorts のURL判定
- キャプション貼り付けからの材料・調理方法の抽出
- 保存済みレシピの検索
- 保存済みレシピの編集・削除
- 家族の星評価と「次はいつ頃食べたいか」の記録
- 評価履歴の削除
- 家族メンバーの追加・名称変更・削除
- 実際の日付での保存・評価記録
- データの書き出し／読み込み（JSONバックアップ）
- ブラウザのlocalStorageへの保存

GitHub Pagesだけでは、SNS URLからキャプション本文を自動取得することや、複数端末間のデータ同期はできません。
これらには将来的にバックエンド/API連携が必要です。
